\partofs{Adjectives}\means{Turns an adjective into its opposite.  Use only if the opposite doesn't already exist.  For adjectives with le- the ke merges to form kel- unless the result would be an illegal consonant cluster.  If the latter case would occur, it stays as kele-.}\example \worddef{lì'u}{word} $\rightarrow$ \worddef{lì'uvi\fixattach{hu} lì'u}{Affixed Word} | \worddef{natkenong lì'ukìng \emph{lì'u\fixattach{hu}}}{Example Sentence with the Word} \source{naviteri.org}{Source Page}\par
